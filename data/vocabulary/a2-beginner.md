# A2 Beginner Vocabulary

## Greetings & Politeness
| Word/Phrase | Meaning | Example |
|-------------|---------|---------|
| hi / hello | casual / formal greeting | "Hi! How are you?" |
| good morning | morning greeting | "Good morning! Did you sleep well?" |
| please | polite addition | "Coffee, please." |
| thanks / thank you | casual / formal thanks | "Thanks so much!" |
| you're welcome | response to thanks | "You're welcome!" |
| excuse me | getting attention | "Excuse me, where's the bathroom?" |
| sorry | apology or to get past someone | "Sorry, can I squeeze by?" |
| bye / goodbye | casual / formal farewell | "Bye! See you tomorrow." |

## Basic Needs & Directions
| Word/Phrase | Meaning | Example |
|-------------|---------|---------|
| bathroom | toilet / restroom | "Where's the bathroom?" |
| how much | asking price | "How much is this?" |
| where is | asking location | "Where is the train station?" |
| here / there | close / far | "Put it here, not there." |
| left / right | directions | "Turn left at the corner." |
| straight ahead | forward direction | "Go straight ahead for two blocks." |

## Numbers & Time
| Word/Phrase | Meaning | Example |
|-------------|---------|---------|
| how many | asking quantity | "How many do you need?" |
| today / tomorrow / yesterday | basic time words | "Let's meet tomorrow." |
| now / later | immediate / delayed | "Can we talk now?" |
| time | asking the hour | "What time is it?" |

## Food & Drink
| Word/Phrase | Meaning | Example |
|-------------|---------|---------|
| water | basic drink | "Can I get some water?" |
| coffee / tea | common drinks | "I'll have a coffee, please." |
| menu | list of food | "Could we see the menu?" |
| bill / check | payment request | "Can I get the check?" |
| hot / cold | temperature | "I'd like an iced coffee — cold." |
